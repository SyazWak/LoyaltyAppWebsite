The Member Experience

A modern, cross-platform loyalty app built with Flutter, Supabase, and Firebase. Members enjoy a premium digital membership card experience with personalized greetings, real-time point tracking, and seamless in-store redemption through secure QR codes.

Core Features

  • Personalized Home Dashboard ‚Äî Dynamic greeting with user's name, prominent points card with tier badge (Bronze/Silver/Gold/Platinum), and a full-width carousel showcasing active campaigns, news, and promotions.
  • QR-Based Point Earning ‚Äî Generate a secure 5-minute QR code at checkout for staff to scan. Real-time validation ensures points are awarded instantly, with automatic refresh of balance and transaction history.
  • Smart Rewards Catalog ‚Äî Browse all available rewards with clear point requirements. Rewards dynamically show as "unlocked" with vibrant gradients when you have enough points, or "locked" with helpful hints on how many more points you need.
  • Secure In-Store Redemption ‚Äî Select a reward, confirm redemption, and receive a unique QR code with a secure token. Staff scan to verify and complete the redemption, with immediate point deduction and receipt confirmation.
  • Comprehensive Transaction History ‚Äî Timeline view of all point activities with color-coded entries (green for earned, orange for redeemed), summary cards showing total earned/redeemed, and filters to view specific transaction types.
  • News & Promotions Hub ‚Äî Stay updated with campaign announcements, limited-time offers, and program updates. Unread count badge on tab, with articles marked as read when opened.
  • Profile Management ‚Äî View and edit personal information, see membership tier with visual badges, check "Member Since" date, review complete transaction history, and securely sign out.
  • Push Notifications ‚Äî Firebase Cloud Messaging integration for real-time alerts about earned points, tier upgrades, expiring rewards, and new promotions.

User Experience Highlights

  • Material 3 Design ‚Äî Modern Flutter UI with gradient backgrounds, smooth animations, rounded cards, subtle shadows, and responsive layouts optimized for real mobile usage.
  • Tier Progression Visuals ‚Äî Color-coded badges and cards that dynamically reflect your membership status, creating a premium "digital membership card" feel.
  • Graceful State Handling ‚Äî Built-in loading states, empty states, and friendly error messages throughout the app for a polished user experience.
  • Cross-Platform Ready ‚Äî Compiled for iOS, Android, web, and desktop from a single Flutter codebase.

üíº SW Loyalty Staff
The Operational Engine

A focused Flutter staff portal designed specifically for brick-and-mortar shops. This "Shopkeeper Portal" gives front-line staff a clean, secure interface to award loyalty points, redeem rewards via QR scanning, and review real-time transaction history‚Äîall backed by Supabase for authentication and data integrity.

Core Capabilities

  • Secure Staff Authentication ‚Äî Email and password login via Supabase Auth with robust error handling and inline feedback. Only authorized staff can access the portal.
  • Modern Dashboard ‚Äî Personalized greeting using staff email, gradient-based card layout, and three primary quick actions: Award Points, Redeem Reward, and Transaction History.
  • Award Points Flow ‚Äî Camera-based QR scanner using mobile_scanner. Scan customer's earn code, validate token on server, prompt for point amount (with helpful guidance like "Typical amounts: 10, 20, 50"), and award points instantly via Supabase RPC function. Handles expired codes, daily limits, and invalid tokens with clear error messages.
  • Redeem Rewards Flow ‚Äî Dedicated QR scanner for redemption codes. Fetches reward details and token status from Supabase, validates token (checks for expired, already-redeemed, or cancelled status), displays rich confirmation dialog with reward title and point cost, and finalizes redemption via server RPC with loading indicator and success confirmation.
  • Fraud Prevention Built-In ‚Äî Real-time server-side validation for every scan, token expiration enforcement, one-time use verification, and daily earning limits to prevent abuse.
  • Transaction History & Analytics ‚Äî Summary header showing total transactions, earned vs. redeemed split, and detailed card-based list with customer email, reward title, point amount (+/-), and human-friendly timestamps. Pull-to-refresh for latest data.

Technical Details

  • High-Performance Scanning ‚Äî Optimized for challenging lighting and fast-paced checkout flows. Auto-restart scanner after each operation for rapid processing of multiple customers.
  • Server-Side Business Logic ‚Äî Sensitive operations encapsulated in Supabase RPC functions (redeem_earn_qr, redeem_redemption_token) to prevent client-side manipulation.
  • Staff-Specific Filtering ‚Äî Transaction history can be filtered by staff_id so employees see their own activity for accountability.
  • Comprehensive Error Handling ‚Äî Friendly, staff-oriented error messages instead of raw technical errors. Loading states with semi-transparent overlays to prevent duplicate scans.

Dashboard & Navigation

Technical Highlights.

Secure QR Logic

Implemented a sophisticated token rotation system where QR codes expire every 5 minutes for earning and 15 minutes for redemption. This multi-layered approach uses short-lived JWT-style tokens to prevent fraudulent screenshots or unauthorized reuse, ensuring that every point earned is verified through a live interaction.

‚ö°

Supabase Real-time Sync

Leveraging WebSocket connections via Supabase to push database changes instantly to all client devices. When a staff member scanner confirms a transaction, the customer's interface updates within milliseconds, eliminating the need for manual refreshes or long-polling.

Dynamic Tiering Engine

Developed a logic-driven progression system that automatically recalibrates user status. Customers move through Bronze, Silver, Gold, and Platinum ranks based on real-time activity metrics, with the UI dynamically adjusting its design language to reflect their new status.

Relational Data Integrity

Architected a transparent audit trail where every transaction is an atomic database operation. Each record is intricately linked to specific staff IDs, timestamps, and customer profiles, providing a bulletproof ledger for system compliance and financial oversight.
